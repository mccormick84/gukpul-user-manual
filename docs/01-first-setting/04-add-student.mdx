# 학생 추가 및 입반
수강생 정보를 시스템에 추가하고 학급에 입반 시키는 과정을 알아봅니다.

:::info 메뉴 찾아가기
__`기본메뉴` → `학급관리` → `학급관리`__
:::

## 학생 추가하기
### 입력 준비
우측 상단의 `추가` 버튼을 눌러 입력 행을 추가합니다.
![입력준비](./img/03-학생추가/학생추가-1.png)

### 학생 정보 입력
학생의 신상 정보를 입력합니다. <span style={{color: "red"}}>`*`</span>로 표시된 필수 정보는 반드시 입력해야 합니다.<br/>
자동 생성되는 정보가 있으므로 첫 입력 시에는 아래의 `개별 항목 안내`를 자세히 읽어주세요.

#### 개별 항목 안내
- 학생명 : <span style={{color: "red"}}>`필수`</span> 학생의 이름 및 성별
- 학년 : 학년을 선택 (`코드관리` - `학교구분/학년` 에서 지정된 값)
- 생년월일 : 학생의 생년월일
- 학교명 : 재학 중인 학교명
- 학부모HP : <span style={{color: "red"}}>`필수`</span> 학부모님의 휴대전화 번호
- 학생HP : 학생의 휴대전화 번호
- 최초등록일 : 학원에 최초 등록 한 날짜
- 학번 : __`자동생성`__ 부모의 휴대전화 끝 4자리 + 임의의 숫자(0~9)
- 할인사유, 할인율 : 수강료 할인 항목과 할인율을 지정
- 형제정보: __`자동매칭`__ 학부모의 휴대전화 번호와 동일한 학생 정보를 표시
- 현금영수증NO : 현금영수증 발급에 사용할 전화번호를 입력 (개인/사업자 선택 가능)
- 아이디 : __`학생HP 입력시 자동 입력`__ 홈페이지에 접속하기 위한 학생 아이디

### 저장 및 수정
필수 정보를 입력 하고 `저장`을 누르면 학생 데이터가 저장되고 `비원생`으로 등록이 됩니다.
![학생추가-2](./img/03-학생추가/학생추가-2.png)

:::tip 사용 팁 : 학생의 상태
* 수강 이력 :
- 신입생 : 직전 달 수강 이력이 없는 학생 (달 기준)
- 재원생 : 현재 학급에 배정되어 있는 경우 + 직전 달 수강이력이 있고(신입생x)
- 퇴원생 : 모든 수강 이력이 종료 된 학생
- 휴원생 : 휴원 처리
- 비원생 : 수강 이력 없는 학생
:::

저장 이후에는 학생을 검색하고 수정할 수 있습니다. <br/>
학생 검색에 대한 자세한 내용은 [링크 추가 예정](#)에서 확인할 수 있습니다.

### 기본 정보 전체보기 (추가 정보 입력)
신상정보의 비밀번호의 오른쪽에 위치한 __`기본정보 전체보기`__를 누르면 학생의 추가 정보를 관리할 수 있습니다.
- 학부모명 : 학부모님의 이름 및 유형(부, 모)을 지정
- 첫 수업 : 학생의 첫 수업 날짜를 기록
- 추가 연락처: 추가로 등록해야 할 연락처가 있을 경우 기입. 문자 발송 여부 선택 가능
- 클리닉 : 학생에게 배정된 클리닉의 요일과 시간을 지정할 수 있습니다. (1개)
- 이메일 : 학생의 이메일 주소
- 닉네임 : 학생에게 별도의 닉네임을 부여할 수 있습니다.
- 우편번호, 주소 : 학생의 주소 정보
- 승차위치 : 차량 이용 정보를 등록 기타기능안내 - 추후 링크 추가 예정
- MEMO : 개인 학생 별 메모 공간입니다. `메모추가`를 이용해 입력 시 기록 날짜와 입력한 사람이 자동 추가 됩니다.
