# 학생 입반 시키기

수업을 진행할 수 있도록 학생을 학급에 입반시키는 작업을 진행합니다. <br/>
수업 진행 및 결과 기록에 대한 자세한 내용은 **[링크추가 예정](#)** 의 내용을 참고 해주시면 됩니다.

## 학급에 학생을 추가

학급을 기준으로 하여 학생을 입반시키는 방법입니다.

:::info 메뉴 찾아가기
`기본메뉴` → `학급관리` → **`학급관리`**
:::

## 개별 학생을 선택하여 학급에 추가

개별 학생을 선택하여 특정 학급에 입반 시키는 방법입니다.

:::info 메뉴 찾아가기
`기본메뉴` → `학생관리` → **`학생관리`**
:::
